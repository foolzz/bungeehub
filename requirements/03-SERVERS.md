## Implementation Notes

### Server List
- **Real-time updates** via WebSocket for status changes
- **Sortable columns** with persistent sort preference
- **Multi-select** with keyboard shortcuts (Shift+Click, Ctrl+Click)
- **Inline actions** for quick operations
- **Batch operations** with confirmation dialogs
- **Export** to CSV/JSON formats

### Server Detail
- **Tab navigation** with URL routing (/servers/hub-1/performance)
- **Auto-refresh** for real-time data (configurable interval)
- **Responsive charts** using Recharts
- **WebSocket** for live updates (player joins/leaves, alerts)
- **Lazy loading** for heavy tabs (logs, files)

### Add/Edit Form
- **Multi-step wizard** with progress indicator
- **Form validation** with real-time feedback
- **Test connection** before saving
- **Auto-save to draft** (localStorage)
- **Template support** for common configurations

### Performance Considerations
- **Pagination** for large server lists
- **Virtual scrolling** for logs
- **Debounced search** to reduce API calls
- **Cached data** with SWR or React Query
- **Lazy load charts** and heavy components
