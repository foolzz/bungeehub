## Implementation Notes

### Player List
- **Real-time updates** for online/offline status
- **Advanced search** with fuzzy matching, UUID, IP
- **Bulk operations** with permission checks
- **Export** player data (CSV/JSON/Excel)
- **Custom columns** - users can show/hide columns
- **Saved filters** - store frequently used filters

### Player Profile
- **Tabbed interface** with lazy loading
- **Real-time session** updates via WebSocket
- **Responsive charts** for statistics
- **Audit trail** for all admin actions
- **Quick actions** toolbar always visible

### Permissions System
- **Inheritance visualization** with tree view
- **Permission tester** to check effective perms
- **Bulk permission** operations
- **Server-specific** permissions support
- **Timed permissions** with auto-expire

### Punishment System
- **Templates** for common punishments
- **Appeal system** integration
- **Escalation** rules (auto-increase duration)
- **Punishment history** affects severity
- **Evidence upload** (screenshots, logs)

### Performance
- **Virtual scrolling** for large lists
- **Lazy loading** player skins
- **Cached player** data with SWR
- **Optimistic updates** for instant feedback
- **Background sync** for heavy operations
